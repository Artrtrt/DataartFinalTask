Запуск проекта:
1)Скачать IntellijIdea Ultimate
2)Скачать jdk(версия 14.0.2)
3)Cкачать Apache TomCat(версия 9.0.54)
4)Заходим в Idea, нажимаем на run -> Edit Configurations, нажимаем + слева сверху и выбираем Tomcat local,
потом слева от Application server нажимаем кнопку Configure , дальше нажимаем + слева сверху.
Появится окно, где нужно выбрать папку с установленным TomCat. Справа от Tomcat Home на окно ввода 
и выбираем папку куда расппаковали TomCat, подтверждаем.
5)Если появилась кнопка Fix снизу окна конфигурации, жмём на неё и выбираем finalTask:war exploded. 
После, снизу в строке Application context выбираем пустое поле.
Жмём apply и закрываем это окно.
TomCat готов к работе!
6)В Idea жмем Run -> Run 'Tomcat' и запускаем сервер. Когда страница загрузится переходим по url http://localhost:8080/siteOfArticles
Проект запущен!
